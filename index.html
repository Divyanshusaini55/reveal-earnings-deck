import pandas as pd
import numpy as np

# Basic EPS calculation example
def basic_eps(net_income, preferred_dividends, weighted_avg_shares):
    return (net_income - preferred_dividends) / weighted_avg_shares

print(basic_eps(4_800_000_000, 0, 361_600_000))  # -> 13.27

Financial Formulas (Math)

Present Value of a cash flow: \[ PV = \frac{CF_t}{(1 + r)^t} \]

Net Present Value over *n* periods: \[ NPV = \sum_{t=0}^{n} \frac{CF_t}{(1+r)^t} \]

Cost of Equity (CAPM): \[ k_e = r_f + \beta (r_m - r_f) \]
